# Instructions for creating log file

File name: _update_log.md

## Template:

```markdown
# Update log

Log of updates.
---
## "Date of update"

### Edited/Deleted/Created file_1.md
**Update based on:** "*_documentation.md" or "changelog patch:" "patch_number")"

*Short description of change made*

### Edited/Deleted/Created file_2.md
...

---
## "Date of update"
...

```

All updates on the same date go under the same header, only make a new header when date is different.
